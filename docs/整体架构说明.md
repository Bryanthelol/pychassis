# 整体架构说明

## 边界层

网关，端点，聚合

- 网关

     - 分发请求，管理请求
       - 限速限流
       - 黑白名单
       - 日志

- Web 服务

     - 接收请求，对接内部微服务

## 服务层

业务逻辑，微服务

- 内部微服务
    
     - 服务间通信
     - 业务逻辑处理
     - 管理所属的数据库


## 平台层

基础设施，通信，部署

- 可观测

     - Metrics 度量
     - Tracing 追踪
     - Logging 记录
